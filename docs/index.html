How it works

Designed to stay
out of your way.

01
Install from Chrome Web Store

One click, no configuration. Polish lives in your sidebar — always there, never in the way.

02
Open the panel on any site

Click the Polish icon in your toolbar. The side panel opens without interrupting the page.

03
Apply a style — or describe one

Pick a preset, toggle a switch, or type a description like "warm sepia, large text, serif font". Polish handles the CSS.

POST /api/generate → /* custom CSS */
04
Save it. Reuse it anywhere.

Save your style as a preset and apply it to any site with one click. Your styles, your web.
